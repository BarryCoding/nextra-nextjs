# useConfig

<Callout type='warning'>
  **The `useConfig` hook is made to dynamically configure your project**. See [Theme
  Configuration](/nextra/docs-theme/theme-configuration) for more information about each customizable feature.
</Callout>

The `useConfig` hook returns data from your theme configuration and current page
context.

## Return Values

- `banner`: data about [Banner](/nextra/docs-theme/theme-configuration#banner)
- `chat`: data about [Chat Link](/nextra/docs-theme/theme-configuration#chat-link)
- `darkMode`: a boolean whether the theme is dark or not
- `direction`: a string that defines the current
  [writing direction](/nextra/docs-theme/theme-configuration#writing-direction)
- `docsRepositoryBase`: a link string to the documentation repository
- `editLink`: data about
  [Edit Link](/nextra/docs-theme/theme-configuration#edit-link)
- `feedback`: data about
  [Feedback Link](/nextra/docs-theme/theme-configuration#feedback-link)
- `footer`: data about [Footer](/nextra/docs-theme/theme-configuration#footer)
- `gitTimestamp`: a function that renders the
  [Last Updated Date](/nextra/docs-theme/theme-configuration#last-updated-date)
- `head`: an equivalent to
  [Next.js Head tag](https://nextjs.org/nextra/pages/api-reference/components/head)
- `logo`: a React component for the website's logo
- `logoLink`: a link for the `logo` component
- `navbar`: components related to
  [Navbar](/nextra/docs-theme/theme-configuration#customize-the-navbar)
- `navigation`: options about
  [Navigation](/nextra/docs-theme/theme-configuration#navigation)
- `themeSwitch`: data about
  [Theme Switch](/nextra/docs-theme/theme-configuration#theme-switch)
- `nextThemes`: data about theme configuration (see
  [pacocoursey/next-themes](https://github.com/pacocoursey/next-themes))
- `notFound`: data to display when the page is not found
- `primaryHue`: data about
  [Theme Color](/nextra/docs-theme/theme-configuration#theme-color)
- `primarySaturation`: data about
  [Theme Color](/nextra/docs-theme/theme-configuration#theme-color)
- `project`: data about
  [Project Link](/nextra/docs-theme/theme-configuration#project-link)
- `search`: data about [Search](/nextra/docs-theme/theme-configuration#search)
- `serverSideError`: data to display when a server-side error happens (requires
  `500.mdx` page too)
- `sidebar`: data about [Sidebar](/nextra/docs-theme/theme-configuration#sidebar)
- `toc`: data about
  [Table of Contents Sidebar](/nextra/docs-theme/theme-configuration#toc-sidebar)
- `useNextSeoOptions`: a function that passes data to Next.js SEO component.
  (see [SEO Options](/nextra/docs-theme/theme-configuration#seo-options))
- `flexsearch`: data about search configuration
- `title`: Title about current document
- `frontMatter`: parsed [front matter](https://jekyllrb.com/nextra/front-matter)
  data
